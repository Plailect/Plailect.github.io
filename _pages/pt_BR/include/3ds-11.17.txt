**Firmware 11.17.0 para o 3DS foi lançado recentemente**. Veja aqui o que você deve saber:

- Se o seu console estiver executando *Luma 10.2.1 ou superior*, a atualização é *100% segura*. Você pode verificar a sua versão do Luma segurando (Select) enquanto inicializa o seu console.
- Se o seu console estiver em uma versão do Luma mais antiga, você deve [atualizar o Luma](checking-for-cfw) antes de atualizar o seu console para 11.17.0.
- Se o seu console ainda não tem custom firmware, **NÃO atualize para a versão mais recente** a menos que este guia o instrua.
- Se o seu console não modificado JÁ ESTIVER na versão 11.17.0, este guia foi atualizado em conformidade. Specifically, New3DS-family consoles have a [free software method](installing-boot9strap-(super-skaterhax)), while Old3DS-family consoles will need to use [alternate exploits](alternate-exploits) that rely on external hardware or software.
