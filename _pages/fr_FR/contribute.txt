Nous sommes ravis que vous souhaitiez contribuer à 3ds.hacks.guide ! Voici quelques façons de le faire :

## Maintenance

Ce guide est open source - son dépôt est disponible [ici](https://github.com/hacks-guide/Guide_3DS). Si vous pensez qu'il y a un problème avec le guide, vous pouvez nous le faire savoir en remplissant un rapport d'incident dans l'onglet "Issues", ou soumettre une demande de pull pour que vos modifications soient intégrées dans le guide.

## Traductions

Si vous souhaitez traduire ce guide dans d'autres langues (ou aider à maintenir les traductions à jour), consultez notre [Crowdin](https://crowdin.com/project/3ds-guide).

## Seeding

Certains fichiers sur ce guide ne sont disponibles que sous forme de torrents. Si vous souhaitez seeder ces torrents pour assurer leur disponibilité à l'avenir, vous pouvez utiliser [ce]({{ "rss. ml" | absolute_url }}) flux RSS.

## Dons

Maintenir ce guide (ainsi que nos canaux de support) est avant tout un effort bénévole, mais si ce guide vous a aidé d'une certaine manière, les dons sont grandement appréciés.
