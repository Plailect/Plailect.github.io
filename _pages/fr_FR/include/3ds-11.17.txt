**Le logiciel système version 11.17.0 de la 3DS est sorti récemment**. Voici ce que vous devriez savoir :

- Si votre console fonctionne sous *Luma3DS 10.2.1 ou une version ultérieure*, la mise à jour peut être faite *sans danger*. Vous pouvez vérifier votre version de Luma3DS en maintenant (Select) pendant le démarrage de votre console.
- Si votre console est sur une version antérieure de Luma3DS, vous devez [mettre à jour Luma3DS](checking-for-cfw) avant de mettre à jour votre console vers la version 11.17.0.
- Si aucun custom firmware n'est installé sur votre console , **ne mettez PAS à jour vers la dernière version**, car les méthodes d'installation peuvent être plus difficiles.
- Si votre console non hackée est DÉJÀ en version 11.17.0, vous pouvez aller sur [cette page](https://troubleshoot.uwuu.ca/seventeen/) (la page sera en anglais) pour voir si une méthode est disponible pour votre console.
- Les informations à propos du logiciel système version 11.17.0 évoluent rapidement. Pour obtenir les dernières informations, rejoignez [Nintendo Homebrew sur Discord](https://discord.gg/MWxPgEp).
